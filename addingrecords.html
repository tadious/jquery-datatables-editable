Editable DataTable example

Preamble

DataTable Editable plugin enchances standard DataTables plugin by adding functionalities that enables a user to edit cells inline, delete rows or add new records. All required processing is encaplulated it the plugin and only thing that is required it to create a server-side code that accepts add, update and delete requests.

To enable adding new record you will need to create a custom HTML form with id "formAddNewRow" where a user can enter information about the new record. When Editable plugin detects add new record form it will generate add button into the "add_delete_toolbar"

On the server-side you will need to create one page that handles delete AJAX requests (I called it DeleteData.php). This page should accept parameter id - id of the record that is deleted (id is placed in tag that surroumds editing cell), and return string "ok" if record is successfully deleted or error message as a string.
Also you will need to create one page that handles update AJAX requests (I called it UpdateData.php). This page should accept following parameters:

  • value - that contains text used entered in the table cell while editing
  • id - id of the record that is deleted (id is placed in tag that surroumds editing cell)
  • columnId - position of the column of the editing cell (hidden columns are counted also)
  • columnPosition - position of the column of the editing cell (hidden columns are not counted)
  • rowId - id of the row containing cell that is edited
Page should return string "ok" is record is successfully deleted or error message as a string.

Initialisation code

$(document).ready( function () {
		$('#example').dataTable().makeEditable({
							sUpdateURL: "UpdateData.php",
							sAddURL: "AddData.php",
                    					sDeleteURL: "DeleteData.php"
							});
}
